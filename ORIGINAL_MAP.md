# Original Drunk Simulator Map (32×57)

This file preserves the original map design before expanding to 40×70 for QR code optimization.

## Map Dimensions
- **Columns**: 32
- **Rows**: 57
- **Total Tiles**: 1,824

## Tile Legend
- `0`: Street (grey)
- `1`: Bar Floor (tan)
- `2`: Patio (light grey)
- `3`: Wall (brown)
- `4`: Door (black - passable)
- `5`: Bar Counter (dark brown)
- `6`: Staff Zone (blue)
- `7`: Stairs (grey-brown)
- `8`: Player Start (green marker)
- `9`: Beer Tap (yellow)
- `10`: Employee Spawn (red)
- `11`: Patron Spawn (orange)
- `12`: Camera Start (cyan)
- `13`: POI - Point of Interest (purple)
- `14`: Chair (dark forest green)
- `15`: Cash Register (money green)

- Cash registers for bartender transactions
